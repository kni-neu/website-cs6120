# Lab 2: Tokenization and Regex Practice

## Overview
This lab focuses on advanced text processing using Regular Expressions and modern tokenizers.

## Tasks
1. **Regex Mastery:** Write patterns for email extraction, phone numbers, and hashtag detection.
2. **Subword Tokenization:** Explore BPE (Byte Pair Encoding) and WordPiece tokenization using the `transformers` library.
3. **Normalization:** Implement custom normalization pipelines.
