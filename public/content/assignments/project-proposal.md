# Project Proposal: NLP Application

## Overview
The project proposal is your first step towards building a successful NLP application. It should clearly define the problem you are solving, the data you will use, and your proposed methodology.

## Requirements
Your proposal should cover the following points:
1. **Application Idea:** What are you building? Why is it useful or interesting?
2. **Data Source:** Where will your data come from? (e.g., specific web-scraped dataset, public corpus, or your own collection).
3. **NLP Tasks:** What specific tasks will your app perform? (e.g., classification, summarization, entity extraction).
4. **Proposed Architecture:** High-level description of the models and tools you plan to use (e.g., LLMs, vector dbs, RAG).

## Submission
* **Format:** PDF (2 pages max)
* **Platform:** Submit via Canvas/Gradescope as per instructions on the main assignments page.
