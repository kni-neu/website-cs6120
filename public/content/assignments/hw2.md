# HW 2: Neural Sentiment Analysis

## Overview
In this assignment, you will build a sentiment classifier using neural networks. You will explore different architectures, from simple MLPs to RNNs.

## Objectives
1. Implement a Bag-of-Words (BoW) neural classifier.
2. Implement a Recurrent Neural Network (RNN) based classifier.
3. Compare performance between traditional and neural methods.

## Data
We will use the IMDb movie review dataset.
